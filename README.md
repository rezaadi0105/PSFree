# PSFree version 1.5.0

PSFree is a collection of exploits for the PS4 console. The main focus of the 
repo is for the PS4 but we try to make things portable to PS5.

* Exploits
  * PSFree: psfree.mjs
  * Lapse (kernel): lapse.mjs
